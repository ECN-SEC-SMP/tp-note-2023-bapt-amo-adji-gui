Add a bottom left corner to a specific box at coordinates [x, y].

This function checks all the nearby walls that could touch the bottom left corner It ensures that those walls are not conflicting and if so it adds the corner.
This function add as well the common walls between boxes by calling "addOneWall()" function once all the checks are done and the corner is ready to be added

Parameters
[in]boxBoardThe 2d array board that contains all the boxes
[in]xThe x coordinate of the box
[in]yThe y coordinate of the box
Returns
true The corner is valid and has been added
false The corner is unvalid (e.g. touching a wall) and has not been added

This function add a corner to a box, depending on the type of the corner.

This function ensures that we are able to put the corner. It checks that the corner type does not touch any nearby walls. If so it will return an error.

Parameters

bool addOneWall ( Box(*)  boxBoard[BOARD_SIZE],
Direction  dir,
int  x,
int  y 
)

This function add a wall at specific [x, y] coordinates.

Since two boxes are sharing a common wall we have to add the "other side" of the wall as well. The edge cases are for the edge of the board were there are no side walls to share (e.g. for the top row there is no upper box, and so on).

Parameters
[in]boxBoardThe 2d array board that contains all the boxes
[in]dirThe direction of the wall (e.g. [top, bottom, left, right])
[in]xThe x coordinate of the box
[in]yThe y coordinate of the box
Return values
trueThe wall was added succesfuly
falsex or y is not in range [0; BOARD_SIZE]
